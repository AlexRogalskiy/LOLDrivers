# procexp.Sys ![:inline](/images/twitter_verified.png) 


### Description

The method of abusing the Process Explorer driver to bypass EDR systems isnâ€™t new; it was implemented in the open-source tool Backstab, first published in June 2021. AuKill drops a driver named PROCEXP.SYS (from the release version 16.32 of process Explorer) into the C:\Windows\System32\drivers path. The legitimate Process Explorer driver is named PROCEXP152.sys, and normally is found in the same location.
